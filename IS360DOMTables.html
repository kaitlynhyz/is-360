Example to Create a New Table

There is a bit more to creating a table because there are many more elements to create and they must be connected together in the DOM tree. There are two ways to do it. Either call the general createElement() method or use the table specific elements of HTMLTableElement.

Create create a table and set a style and attribute.

Create a Table using General Tags

            var tbl = document.createElement('table');
            tbl.style.width = '50%';
            tbl.setAttribute('border', '1');

Let's assume that I am using a <tbody> tag. So i create it.

            var tbdy = document.createElement('tbody');

Then, for each row, i create the <tr> tag using a for loop.. And inner for loop creates the cells. 

            // The outer loop creates the rows.
            for (var rows = 0; rows < 3; rows++) {
                var tr = document.createElement('tr');

                // The inner loop creates the cells.
                for (var cells = 0; cells < 2; cells++) {
                    {
                        // Create the cell.
                        var td = document.createElement('td');
                        // Create the Text node that will be a child of the cell.
                        td.appendChild(document.createTextNode('HELLO'))
                        // Append the cell to the row.
                        tr.appendChild(td)
                    }
                }
                // Append the row to the table body.
                tbdy.appendChild(tr);
            }

Note that I add the text node to the cell and the cell to the row. After the cells have been added, the row is added to the body.

The next statement adds the body to the table.

            tbl.appendChild(tbdy);

And finally the table is added to a <div> tag placeholder.
