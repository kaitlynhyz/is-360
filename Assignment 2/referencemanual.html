Abstract

The anchor a tag defines a hyperlink. When the anchor tag is used with its href attribute, it creates a hyperlink to webpages, files, email address, and/or locations on the same page of the website. Anchor tags can be used to create links to other pages and bookmarks to different sections of the page.

Details

Attributes applicable to the anchor tag:

  • href - Specifies the URL of the page the link goes to. If the href attribute is not present, the a tag will not be a hyperlink
    • tel: - Used to initiate a phone call with a link
    • mailto: - Activates the default mail client on the computer for sending an e-mail
    • #section - Sections of a page within fragment URLs
  • download - Specifies that the file will be downloaded when a user clicks on the hyperlink

Examples

Absolute references

An absolute reference is used to refer to a site somewhere else on the internet. The absolute path contains a complete URL to the website.

  1. a href = "http://www.google.com"
  2. a href = "http://www.linkedin.com/in/kaitlynzhang"

Relative references

A relative reference points to a file within a website. A relative link only includes the name of a specific file or page to the current path.

  1. a href = "index.html"
  2. a href = "../referencemanual.html"
